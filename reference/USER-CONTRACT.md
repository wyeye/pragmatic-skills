# User contract

Users ask for work normally. They do not need to know internal skill names.

The agent starts at the PSP entry, uses a direct route only for explicit instruction maintenance or explicit workflow retrospective, otherwise triages into one primary mode, and loads support skills only when the current phase needs them.

The agent must distinguish requirement confirmation from safety approval, preserve user-owned work, attach evidence to strong claims, and disclose missing verification or residual risk.
